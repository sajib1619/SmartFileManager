# Smart File Manager with Git Integration

## Features
- File operations (copy, move, delete, rename)
- Search system
- Backup system
- Permission manager
- Git integration (init, commit, log, restore)
- GitHub push/pull

## Tools Used
- Bash scripting
- Zenity GUI
- Git
